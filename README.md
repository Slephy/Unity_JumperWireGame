## コントローラーがない時の遊び方
1. スタート画面で「T」キーを押す。すると、コントローラーが接続されていなくともスタートボタンを押せるようになる。
2. コントローラーの代わりにキーボードを用いて操作する。
例えば「Z」「C」を続けて押すと、最も手前のコネクタから赤色のバケツへ伸びるパイプが生成される。もう一度同じ操作をすると、今生成したパイプが削除される。コネクタ側の指定に「Ｚ」「Ａ」「Ｑ」、バケツ側の指定に「Ｃ」「Ｄ」「Ｅ」を用いる。
