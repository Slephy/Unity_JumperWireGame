## 遊び方
本来は Arduino で作ったコントローラーで遊ぶことを想定したゲームですが、以下の操作方法によってキーボードでも遊ぶことが出来ます。

使用するキーは `Z,A,Q,C,D,E` の6種類です。`Z,A,Q` が3つのコネクタに、`C,D,E` が3つのバケツに対応します。
例えば `Z→D` のようにそれぞれコネクタとバケツに対応するボタンを続けて押すと、**その2つを繋ぐパイプが生成されます。** **もう一度同じ入力をすると、パイプが削除されます。**
